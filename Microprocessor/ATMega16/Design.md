# 微机原理 ATMega16 课程设计

## 准备工作

### 环境要求

- 硬件环境：ATMega16A 开发板 1 个，ISP 数据线 1 条。

- 软件环境：Windows 操作系统，PROGISP 1.72 烧写软件，ICCAVR 7.22 编译软件。

### 软件配置

- PROGISP：
  - 需要设置 Select Chip 为 ATMega16A。

- ICCAVR：
  - 需要设置 Compiler Options - Target - Device Configuration 为 ATMega16。
  - 需要设置 Compiler Options - Paths - Include Paths 为 ICCACR 软件安装目录的 include 文件夹。
  - 需要复制 [AVR_HJ-2G.H](AVR_HJ-2G.H) 文件到 ICCACR 软件安装目录的 include 文件夹。

- 文本编辑器：
  - 需要采用 GBK 编码。

## 设计内容

主题：智能摇头风扇

### 先行测试

目的：测试红外传感器的灵敏度。

零件：

- 人体红外传感器 1 个（输出数字高电平）
- 板载 LED 1个

预期行为：

- 当红外传感器检测到人体时，板载 LED 点亮，否则熄灭。

接口规定与连接：

- 红外传感器输出口连接 PD0 引脚；
- 板载 LED 使用 PB7 引脚。

代码：design0.c

### 第一版

零件：

- SG90 舵机 1 个
- 集成风扇电机模块 1 个（含有 INA、INB 两个控制口）
- 人体红外传感器 2 个（有效输出为数字低电平）
- 按钮 1 个
- 多输出电源模块 1 个（输出 5V 供舵机和电机使用，输出 3.3V 供单片机使用）

预期行为：

- 当任意红外传感器检测到人体，风扇工作；
  - 当左侧红外传感器检测到人体时，风扇向左摇头；
  - 当右侧红外传感器检测到人体时，风扇向右摇头；
  - 当两个红外传感器同时检测到人体时，风扇左右周期性摇头；
- 当两个红外传感器都没有检测到人体时，风扇停止工作。

- 按下按钮后，循环切换风扇低速、中速、高速状态。

接口规定与连接：

- 舵机控制口连接 PB0 引脚；
- 风扇电机控制口 INA 连接 PB2 引脚，INB 连接 PB1 引脚；
- 左侧红外传感器输出口连接 PD1 引脚，右侧红外传感器输出口连接 PD2 引脚。
- 按钮连接 PD3 引脚。

<summary>详细连接方式：</summary>
根据实际板子布局，具体连接方式如下：

- 220 V 交流电源 -> 多路电压输出模块
- 多路电压输出模块 3.3V -> ATMega16 VCC 10 引脚、两个红外传感器 VCC 引脚
- 多路电压输出模块 5V -> 舵机 VCC 引脚、电机 VCC 引脚
- 共接地 ATMega16 GND 11 引脚、两个红外传感器 GND 引脚、舵机 GND 引脚、电机 GND 引脚、板载按钮
- 舵机 IN 控制口 -> ATMega16 PB0 1 引脚
- 电机 INA 控制口 -> ATMega16 PB2 3 引脚
- 电机 INB 控制口 -> ATMega16 PB1 2 引脚
- 左侧红外传感器 OUT 信号口 -> ATMega16 PD1 15 引脚
- 右侧红外传感器 OUT 信号口 -> ATMega16 PD2 16 引脚
- 板载按钮（电键） -> ATMega16 PD3 17 引脚
