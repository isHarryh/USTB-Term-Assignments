# AGENT.md

此文件夹是**自动控制原理实验课程**的工作区。

你作为一个智能体，可能被赋予以下任务：

- 根据 README.md 中的*实验要求*，遵循其示例风格，完成*实验任务*（主要是 MATLAB 代码生成和数学分析工作）；
- 根据用户的要求，修复、调整或解释*实验任务*；
- 根据用户的要求，生成*实验报告*或总结。

如需生成*实验报告*，它应按照实验题目来分别组织，每个实验题目下，需包括不限于以下关键信息：
1. 实验目的；
2. 参数设置：列出关键模块的参数（如传递函数的系数、初始参数、求解器类型及步长），若进行了多组对比实验，请列表说明各组参数的差异；
3. 描述仿真执行流程；
4. 实验结果及讨论：给出实验图片，要求图片清晰，连线整齐；
5. 实验结论。

需要特别注意：

1. 如果*实验要求*中涉及图片（例如曲线图、图表或结构框图），但你无法看到图片的具体内容，请你务必询问用户：“我发现该实验中有关于某某图像的细节，但是图像在文档中缺失/我现在无法读取图像的内容。请你向我描述图像的内容，以便我继续分析。”
2. 需要明确告知用户每个*实验任务*需要提交或补充的材料，比如，你可以对用户说：“请注意，该项实验任务要求提交数值结果、某某图表和某某界面的截图。”
3. 在撰写*实验报告*时，如果你判断需要插入命令输出或相关图表，你需要在实验报告对应位置写上“TODO：此处需要插入某某内容”。如无特殊要求，*实验报告*需要放在*实验要求*同级目录的 REPORT.md 文件中。
