# 实验三报告：MATLAB 实现控制系统串联校正的频域设计

---

## 综合实践 1：串联超前校正

### 实验目的

对给定单位负反馈系统 $G_0(s)=\frac{K}{s(s+1)}$ 设计串联超前校正装置，使校正后系统满足：
1. 斜坡输入 $r(t)=t$ 时，稳态误差 $e_{ss}\le 0.1$；
2. 开环系统截止频率 $\omega_c\ge 6$ rad/s；
3. 相角裕度 $\gamma\ge 60^\circ$。

### 参数设置

| 项目 | 符号 | 数值 | 说明 |
| :--- | :--- | :--- | :--- |
| 开环增益 | $K$ | 10 | 由 $e_{ss}=1/K_v\le 0.1$ 确定，对 I 型系统 $K_v=K$ |
| 校正前开环传递函数 | $G_0(s)$ | $\frac{10}{s(s+1)}$ | 代入 $K=10$ |
| 超前校正装置 | $G_c(s)$ | $\frac{0.6159s+1}{0.04422s+1}$ | 校正网络传递函数 |
| 校正网络零点 | $z_c$ | $-1.624$ | $1/\tau=-1/0.6159$ |
| 校正网络极点 | $p_c$ | $-22.61$ | $-1/0.04422$ |
| 分度系数 | $\alpha$ | 0.0718 | $0.04422/0.6159$（$\alpha<1$ 确认为超前网络） |
| 最大超前角频率 | $\omega_m$ | $\approx 6.06$ rad/s | $\omega_m=1/\sqrt{\alpha\tau^2}=1/\sqrt{0.6159\times 0.04422}$ |
| 最大超前相位 | $\phi_m$ | $\approx 60^\circ$ | $\phi_m=\arcsin\frac{1-\alpha}{1+\alpha}=\arcsin\frac{0.9282}{1.0718}$ |

### 仿真执行流程

1. **确定 $K=10$**：由斜坡稳态误差要求 $e_{ss}=1/K=0.1$ 得出。
2. **建立校正前模型**：`G0 = tf(10, [1, 1, 0])`，利用 `margin()` 求原系统的幅值裕度、相角裕度和截止频率。
3. **构造校正网络**：`Gc = tf([0.6159, 1], [0.04422, 1])`。
4. **计算校正后开环模型**：`Gk = Gc * G0`。
5. **绘制 Bode 图对比**：校正前（蓝色实线）与校正后（红色虚线）。
6. **计算闭环传递函数**：分别对校正前后求单位负反馈闭环模型 `feedback(G, 1)`。
7. **绘制单位阶跃响应对比**：校正前（蓝色实线）与校正后（红色虚线）。
8. **使用 `find()` 提取时域指标**：超调量、峰值时间、上升时间（10%~90%）、调节时间（2% 准则）。

### 实验结果及讨论

#### 频域分析

校正前系统：
- 静态速度误差系数 $K_v = 10$，满足稳态误差要求。
- 相位裕度 $\gamma_0 \approx 18^\circ$（$G_0(j\omega)$ 在 $\omega_{cp}\approx 3.08$ rad/s 处穿越 0 dB，此时 $\angle G_0 = -90^\circ - \arctan(3.08) \approx -162^\circ$），**不满足** $\gamma\ge 60^\circ$ 的要求。
- 截止频率 $\omega_{cp}\approx 3.08$ rad/s，**不满足** $\omega_c\ge 6$ rad/s 的要求。

校正后系统：
- 超前网络在 $\omega_m\approx 6.06$ rad/s 处提供约 $60^\circ$ 的超前相位，将校正后截止频率推至约 6 rad/s。
- 校正后相位裕度 $\gamma \approx 60^\circ$，**满足设计要求**。

#### 时域分析

校正后系统的瞬态响应显著改善：
- 超前校正加快了系统响应速度，校正后截止频率增大（$\omega_c$ 从 $\approx 3.08$ 提升至 $\approx 6$ rad/s），频带变宽，相当于引入了微分效应。
- 超调量减少（相位裕度从 $18^\circ$ 提升至 $60^\circ$，阻尼增大）。
- 调节时间缩短。

### 实验结论

通过串联超前校正装置 $G_c(s)=\frac{0.6159s+1}{0.04422s+1}$，系统相位裕度从约 $18^\circ$ 提升至约 $60^\circ$，截止频率从约 $3.08$ rad/s 提升至约 $6$ rad/s，满足了所有三项设计指标。超前校正利用其相位超前特性增大了系统的相位裕度，使校正后中频段斜率更加合理（趋于 $-20$ dB/dec），验证了频域法设计串联超前校正的有效性。

## 综合实践 2：串联滞后校正

### 实验目的

对给定单位负反馈系统 $G_0(s)=\frac{K}{s(s+1)(0.25s+1)}$ 设计串联滞后校正装置，使校正后系统满足：
1. 斜坡输入 $r(t)=t$ 时，稳态误差 $e_{ss}\le 0.2$；
2. 相角裕度 $\gamma\ge 45^\circ$。

### 参数设置

| 项目 | 符号 | 数值 | 说明 |
| :--- | :--- | :--- | :--- |
| 开环增益 | $K$ | 5 | 由 $e_{ss}=1/K_v\le 0.2$ 确定，对 I 型系统 $K_v=K$ |
| 校正前开环传递函数 | $G_0(s)$ | $\frac{5}{s(s+1)(0.25s+1)}$ | 代入 $K=5$ |
| 滞后校正装置 | $G_c(s)$ | $\frac{16.39s+1}{113.386s+1}$ | 校正网络传递函数 |
| 校正网络零点 | $z_c$ | $-0.0610$ | $-1/16.39$ |
| 校正网络极点 | $p_c$ | $-0.00882$ | $-1/113.386$ |
| 分度系数 | $\beta$ | 6.917 | $113.386/16.39$（$\beta>1$ 确认为滞后网络） |
| 低频增益衰减 | — | $\approx 16.8$ dB | $20\log_{10}\beta$ |

### 仿真执行流程

1. **确定 $K=5$**：由斜坡稳态误差要求 $e_{ss}=1/K=0.2$ 得出。
2. **建立校正前模型**：`G0 = tf(5, conv(conv([1, 0], [1, 1]), [0.25, 1]))`，利用 `margin()` 求原系统的幅值裕度、相角裕度和截止频率。
3. **构造校正网络**：`Gc = tf([16.39, 1], [113.386, 1])`。
4. **计算校正后开环模型**：`Gk = Gc * G0`。
5. **绘制 Bode 图对比**：校正前（蓝色实线）与校正后（红色虚线）。
6. **计算闭环传递函数**：分别对校正前后求单位负反馈闭环模型 `feedback(G, 1)`。
7. **绘制单位阶跃响应对比**：校正前（蓝色实线）与校正后（红色虚线）。
8. **使用 `find()` 提取时域指标**：超调量、峰值时间、上升时间（10%~90%）、调节时间（2% 准则）。

### 实验结果及讨论

#### 频域分析

校正前系统：
- 静态速度误差系数 $K_v = 5$，满足稳态误差要求。
- 在 $\omega\approx 2$ rad/s 处，$\angle G_0 \approx -180^\circ$ 且 $|G_0|\approx 0\text{ dB}$（即增益穿越频率与相位穿越频率几乎重合），相位裕度约为 $0^\circ$，**系统临界不稳定**，**不满足** $\gamma\ge 45^\circ$ 的要求。

校正后系统：
- 滞后网络具有低通滤波特性，其零极点位于非常低的频率（$z_c=-0.061$, $p_c=-0.00882$），对系统低频段（稳态性能）几乎无影响，$K_v$ 保持为 5。
- 在中高频段，滞后网络引入约 $20\log_{10}\beta=20\log_{10}6.917\approx 16.8$ dB 的幅值衰减，使校正后截止频率减小、相位裕度增大至约 $45^\circ$。
- 校正后相位裕度满足 $\gamma\ge 45^\circ$ 的要求。

#### 时域分析

滞后校正对瞬态响应的影响与超前校正相反：
- 校正后截止频率**减小**，系统频带变窄，瞬态响应速度变慢（上升时间和调节时间可能有所增加）。
- 但系统的相对稳定性得到了根本改善——从不稳定/临界稳定变为具有足够相位裕度的稳定系统。
- 由于滞后网络在低频处引入了零极点对，可能使阶跃响应的尾部收敛变慢（长拖尾现象）。

### 实验结论

通过串联滞后校正装置 $G_c(s)=\frac{16.39s+1}{113.386s+1}$，系统从临界不稳定（$\gamma\approx 0^\circ$）改善为具有约 $45^\circ$ 相位裕度的稳定系统，满足了设计要求。滞后校正利用其高频衰减特性降低截止频率以换取相位裕度的提升，但不影响系统的稳态精度（$K_v=5$ 不变），验证了频域法设计串联滞后校正的有效性。

## 超前校正与滞后校正的比较

| 比较维度 | 综合实践 1（超前校正） | 综合实践 2（滞后校正） |
| :--- | :--- | :--- |
| 校正前系统 | $\frac{10}{s(s+1)}$（二阶，相位裕度不足） | $\frac{5}{s(s+1)(0.25s+1)}$（三阶，临界不稳定） |
| 校正装置 | $G_c(s)=\frac{0.6159s+1}{0.04422s+1}$ | $G_c(s)=\frac{16.39s+1}{113.386s+1}$ |
| 校正后截止频率变化 | 增大（$\approx 3.08\rightarrow 6$ rad/s） | 减小 |
| 校正后相位裕度变化 | $\approx 18^\circ\rightarrow 60^\circ$ | $\approx 0^\circ\rightarrow 45^\circ$ |
| 瞬态响应速度 | 变快（微分效应） | 变慢（低通滤波效应） |
| 高频噪声抑制 | 变差（频带变宽） | 变好（频带变窄） |
| 适用场景 | 稳态精度足够、需加快响应、提高相位裕度 | 稳态精度和相位裕度均需改善、可接受较慢响应 |

串联校正环节对系统稳定性及过渡过程的影响可归纳为：
- **超前校正**通过在中频段提供超前相位来增大相位裕度，使截止频率增大、响应加快，但抗高频噪声能力下降；
- **滞后校正**通过降低中高频段增益来换取相位裕度，截止频率减小、响应变慢，但抗高频噪声能力增强；
- 两种校正方法均不影响系统的稳态精度（低频段特性保持不变）。
